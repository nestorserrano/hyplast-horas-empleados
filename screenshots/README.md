## 📸 Cómo Tomar las Capturas

### Método 1: Snipping Tool (Windows)
1. Presiona **Win + Shift + S**
2. Selecciona el área a capturar
3. Guarda con el nombre correspondiente

### Método 2: ShareX (Recomendado)
1. Descarga [ShareX](https://getsharex.com/)
2. Configura atajos de teclado
3. Captura automáticamente con nombres

### Método 3: Pantalla Completa
1. Presiona **Win + Print Screen**
2. Recorta la imagen necesaria
3. Guarda en esta carpeta

## 🎨 Especificaciones

- **Formato**: PNG (mejor calidad) o JPG (menor tamaño)
- **Resolución**: 1920x1080 o superior
- **Tamaño máximo**: 500KB por imagen
- **Nombres**: Sin espacios, usar guiones (-)
- **Contenido**: Datos de prueba, NO datos reales sensibles
